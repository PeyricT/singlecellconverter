# SinglecellConverter

SinglecellConverter is a tool to convert R Seurat (H5Seurat) objects into Python Anndata (H5ad) objects and vice versa. 

---

## Quickstart 

### Installation

```sh
pip install git+https://github.com/PeyricT/singlecellconverter.git
```

### Commands

*seurat2anndata* and *anndata2seurat*. for help : 

```sh
seurat2anndata --help
anndata2seurat --help
```
